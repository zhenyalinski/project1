PyCHARM PAGE

PyCharm is an amazing IDE which provides essentials tools to any pycharm user

There are three editions available today: Community, Edu and Professional

Pycharm fully support HTML Javascript and even CSS

There are some system requirements: 4GB of RAM, any modern GPU and about 3.5GB of disk space

Opening a project in Pycharm is very easy, just go to the project section and choose from Open/VCS

Pycharm also has Services and Terminal incorporated into it

The Search Everywhere feature is a great tool to search for a code if you need to adjust it

The right-click feature has anything from running to debugging your code

With PyCharm Professional, you can now easily visualize your statical data and see bigger picture

PyCharm also has an R plugin which allows you to use R language to compute your data

PyCharm also allows you to tailor your own environment by changing appearance, editor features etc.

All actions within Pycharm are mapped around keyboard shortcuts, so you can be more productive

Pycharm supports most of major testing frameworks like Doctest, pytest and more

IDE supports the debugger on all well known platforms out there

You can easily navgate into history of changes by tapping Back/Forward navigation

Source: https://www.jetbrains.com/help/pycharm/quick-start-guide.html
