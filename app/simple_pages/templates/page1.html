DOCKER PAGE

Docker is an open source platform that is great for developing and running applications

It is an absolute gem when it comes to saving time because you can ship, test and deploy same time

Whats so unique about Docker is that you can run an application in an isolated environment

By running your application in siolation you can test as much as you can and they are lightweight

When it comes to applications in docker, they are actually containers, which can be viewed easily

Another great feature is that you can easy share your containers with others

All you colleagues have to do is to work on code locally and share it with your teammates

With Docker you will experience reliable and consistent delivery

Docker ia based of the client - server relationship

One of the unique features of Docker is daemon, whoch listens on API and does all the lifting

Users interact with daemon via Docker client carrying out requests like "docker ps" and others

Docker also has the Docker Desktop which is utilized for sharing and building apps aka containers

Containers get build by images which act like a template or guide on what container to build

You can always rely on the "docker ps" command to check up on what containers run for you

Source: https://docs.docker.com/get-started/overview/
